# Práctica Final DFIR - Bootcamp KeepCoding 2026

Este repositorio contiene el informe final de la práctica DFIR realizada en el contexto del Bootcamp de Ciberseguridad KeepCoding 2026.

## Contenido

- `Informe_DFIR_Gabriel_Practica_2026.pdf`: informe final de análisis forense digital.

## Descripción

La práctica se dividió en tres bloques principales:

1. Análisis forense de una imagen Windows.
2. Adquisición y análisis inicial de memoria RAM.
3. Comparación de metadatos en imágenes digitales.

Durante el análisis se trabajó con artefactos como registro de Windows, Prefetch, Papelera de reciclaje, logs EVTX, TeamViewer, hashes NTLM, Volatility 3 y ExifTool.

## Herramientas utilizadas

- FTK Imager
- Registry Explorer
- PowerShell
- Mimikatz en modo offline
- Hashcat
- EvtxECmd
- WinPmem
- Volatility 3
- ExifTool

## Nota

Este repositorio tiene fines académicos. No se incluyen imágenes forenses, volcados de memoria, hives de registro, muestras, credenciales reales ni archivos sensibles derivados del laboratorio.
